# Tic Tac Toe Game
GUI application With Java using swing

## Technologies Used
- JAVA
  - Swing Package
  - AWT Event Package

## Setup
- Code Editor (IDE)
- JDK-8

## Run Locally

- Open Command Prompt .
- Change directory to downloaded project folder.
- Compile java file.
```bash
   javac tic_tac_toe.java
``` 
- After successful compilation run class file.
```bash
   java tic_tac_toe
``` 
- GUI will be displayed.
